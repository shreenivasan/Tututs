Captcha Project with Ajax and Jquery Validation

Dependencies

www.jquery.com
Look for the following scripts and download it to scripts folder(its already present there).if you want the newer version of it.

1.jquery.js
2.jquery.form.js
3.jquery.validate.js

Installation
1.Place the captcha folder in yoor server directory.
2.Check the config.php and alter the parameters as you want.
3.Check the check.js files and alter the file as you want..
4.Download font you prefer and place it in fonts directory.
5.Create a directory called captcha in your server.(for storing generated captcha images)

you can find the documentation about jquery you can look through folleowing website..
